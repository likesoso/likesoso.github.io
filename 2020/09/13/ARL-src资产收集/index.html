ARL-src资产收集

简介

快速侦查和目标关联的互联网资产，构建基础资产信息库。

安装

只支持linux、macos，我们选择用vps加docker的安装方式：

mac安装docker

修改安装源

preference->docker Engine中添加

拉取镜像

修改docker-compose.yml 中services web image 和 services worker image 对应的镜像地址

启动docker：

访问5003 端口即可：

默认用户名密码admin/arlpass
